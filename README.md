## 😃 关于我们 
**这里是中国Web3协会，成立于2023年2月7日**
（也许是，也希望未来是）中国最早的Web3社群
我们建立的初衷是因为创始人烟波,苦于中文互联网中的 Web3 相关知识分享内容相对较少，而为了改变这一现状，希望未来能整合和拓宽这类方向相关的 中文知识内容，从而建立了该组织

**社群宗旨**   
永远关注知识和技术的进步，而不是价格   
在此，我们希望为所有的对Web3未来感兴趣和欲为其“添砖加瓦”的朋友们一起,创造出更美好的Web3未来前景！
